# THÔNG TIN BỐI CẢNH DỰ ÁN

Lưu trữ thông tin cốt lõi dự án. Đảm bảo nhất quán báo cáo.

## 1. Thông tin chung
- **Đề tài:** Chatbot tư vấn tuyển sinh (Virtual Assistant) kiến trúc RAG.
- **Sinh viên:** Trần Hồng Sơn (61TH1).
- **Giảng viên:** TS. Lý Anh Tuấn.
- **Lĩnh vực:** AI, NLP, Chatbot.

## 2. Công nghệ & Nền tảng
- **Platform:** Zalo Bot (người dùng cuối) + Web Admin. Không Webchat.
- **Tech Stack:**
    - **Lang/Frame:** Python, FastAPI.
    - **Orchestration:** LangChain.
    - **Vector DB:** ChromaDB (lưu truy xuất ngữ nghĩa).
    - **LLM/Embed:** Google Gemini API (gemini-3-flash-lite, gemini-embedding-2).
- **Kỹ thuật cốt lõi:** RAG + LLM.
- **Frontend/Giao diện:** Zalo Bot (Không gọi Zalo OA Bot).
- **Nguồn dữ liệu:** Tuyển sinh ĐH Thủy Lợi (điểm chuẩn, quy chế, ngành).
