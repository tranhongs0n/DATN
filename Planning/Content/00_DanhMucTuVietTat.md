# DANH MỤC CÁC TỪ VIẾT TẮT VÀ GIẢI THÍCH CÁC THUẬT NGỮ

## 1. Danh mục từ viết tắt

| Từ viết tắt | Cụm từ đầy đủ tiếng Anh | Giải nghĩa tiếng Việt |
| :--- | :--- | :--- |
| **AI** | Artificial Intelligence | Trí tuệ nhân tạo |
| **API** | Application Programming Interface | Giao diện lập trình ứng dụng |
| **CSTT** | - | Cơ sở tri thức |
| **DB** | Database | Cơ sở dữ liệu |
| **LLM** | Large Language Model | Mô hình ngôn ngữ lớn |
| **NLP** | Natural Language Processing | Xử lý ngôn ngữ tự nhiên |
| **RAG** | Retrieval-Augmented Generation | Kỹ thuật sinh văn bản có tăng cường truy xuất |
| **RLHF** | Reinforcement Learning from Human Feedback | Học tăng cường từ phản hồi của người dùng |
| **UI** | User Interface | Giao diện người dùng |
| **UX** | User Experience | Trải nghiệm người dùng |

## 2. Giải thích thuật ngữ

| Thuật ngữ tiếng Anh | Giải nghĩa tiếng Việt | Mô tả |
| :--- | :--- | :--- |
| **Answer Relevance** | Độ liên quan của câu trả lời | Đánh giá mức độ câu trả lời đi thẳng vào trọng tâm và giải quyết trực tiếp truy vấn của người dùng. |
| **Context Precision** | Độ chuẩn xác ngữ cảnh | Đánh giá tỷ lệ thông tin hữu ích và có giá trị thực sự nằm trong các đoạn tài liệu được truy xuất. |
| **Context Recall** | Độ bao phủ ngữ cảnh | Đánh giá khả năng hệ thống truy xuất đầy đủ tất cả các thông tin cần thiết để giải quyết trọn vẹn truy vấn. |
| **Faithfulness** | Độ trung thành | Đánh giá mức độ câu trả lời bám sát nguồn tài liệu gốc, không tự bịa đặt hay suy diễn sai lệch. |
