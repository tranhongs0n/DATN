# DATN — TLU Admission RAG

Multimodal Retrieval-Augmented Generation system for Thuyloi University (TLU) admission information. FastAPI backend + Chroma vector store + Gemini models, with a chat UI, admin UI, and Zalo OA integration.

## Project layout

```
code/
├── main.py              # CLI entrypoint (ui / scrape / build-db)
├── config.yaml          # Non-secret config (models, scraper, paths)
├── prompts.yaml         # LLM prompt templates
├── .env                 # Secrets (NOT committed) — see .env.example
├── pyproject.toml       # Dependencies / package metadata
├── src/
│   ├── config/          # settings.py — loads config.yaml + .env
│   ├── core/            # RAG engine: vector_db, indexing, chat_service,
│   │                    #   semantic_cache, multimodal, auth, chat_db
│   ├── utils/           # scraper, document_loader, zalo_api, logging, ...
│   └── app/             # FastAPI app
│       ├── api.py       # App factory + router mounting + CORS
│       ├── dependencies.py  # Auth, rate limiting, shared services
│       ├── routers/     # auth, chat, zalo, admin_users, admin_data, admin_scrape
│       └── static/      # Chat UI (index.html) + Admin UI (admin.html) + JS/CSS
├── scripts/             # Maintenance/dev scripts (eval dataset, model listing)
├── tests/               # Evaluation (Ragas) — datasets + eval.py
├── data/                # Source corpus (gitignored — stored externally)
└── chroma_db/           # Vector store (gitignored — regenerated by build-db)
```

## Setup

1. Install (editable):
   ```bash
   pip install -e .
   ```
2. Create `.env` from the template and fill in secrets:
   ```bash
   cp .env.example .env
   ```
   Keys:
   - `GOOGLE_API_KEY` — Gemini API key (required)
   - `ZALO_ACCESS_TOKEN` — Zalo OA token (optional, for Zalo integration)
   - `ALLOWED_ORIGINS` — optional, comma-separated CORS origins (defaults to localhost:8000)
3. Put source documents under `data/` (`DaiHoc/`, `ThacSi/`, `TienSi/`, `AdminUploads/`). Corpus is not committed to git.

## Usage

```bash
# Launch chat + admin UI  (http://localhost:8000 , /admin)
python main.py ui [--port 8000]

# Scrape TLU admission site into data/
python main.py scrape [--limit N]

# Build / rebuild the Chroma vector DB from data/
python main.py build-db [--files f1 f2 ...] [--limit N] [--append]
```

No subcommand → launches the UI on port 8000.

On Windows, `start.bat` (repo root) starts the webhook forwarder + API together.

## Notes

- `data/`, `chroma_db/`, `logs/`, `.env`, and `*.egg-info/` are gitignored. Rebuild the vector DB with `build-db` after pulling.
- Config split: non-secret settings in `config.yaml`, secrets in `.env`.
